# SOC Investigations Portfolio

This repository contains a collection of simulated **Security Operations Center (SOC) investigations** designed to demonstrate practical incident analysis skills used by SOC Analysts.

The goal of this portfolio is to showcase real-world security investigation workflows including alert triage, log analysis, IOC extraction, and MITRE ATT&CK mapping.

---

## Skills Demonstrated

• Alert triage and incident validation  
• Log analysis and event correlation  
• Phishing investigation and email header analysis  
• Brute force attack detection  
• Suspicious login investigation  
• Malware investigation and hash analysis  
• IOC extraction and threat intelligence lookup  
• MITRE ATT&CK technique mapping  
• Incident classification and severity assessment  
• Security response recommendations

---

## Investigation Case Studies

### Phishing Investigation
Analysis of a suspected phishing email including header analysis, malicious URL identification, and IOC extraction.

### Brute Force Attack Detection
Investigation of repeated authentication failures indicating a possible brute force attack against a user account.

### Suspicious Login Investigation
Analysis of abnormal login behavior including geolocation anomalies and impossible travel detection.

### Malware Investigation
Investigation of a malicious file detected on an endpoint including hash reputation checks and threat intelligence analysis.
